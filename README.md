# StudyArc Planner

StudyArc Planner is a complete 12-week study planner for learning ArcGIS Pro, ArcGIS Online and Web GIS concepts. This front-end demo (React + Vite + PostCSS) provides a detailed day-by-day roadmap, trackable progress saved to localStorage, and consolidated resources.

## Features
- 12-week study plan with daily tasks
- Mark tasks done and track overall progress
- Day view with objectives, resources and exercises
- Consolidated resources page
- Dark / Light mode and responsive layout
- Smooth animations with Framer Motion

## Run locally
```bash
git clone https://github.com/saadbarghouth2-blip/studyarc-planner.git
cd studyarc-planner
npm install
npm run dev
```
Open http://localhost:5173

## Deploy to GitHub Pages
```bash
npm run build
npm run deploy
```
